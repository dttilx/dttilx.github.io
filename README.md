# dttilx.github.io

dttilx 的个人作品集网站，展示项目、技术栈和联系方式。

访问地址：

```text
https://dttilx.github.io/
```

## 部署方式

1. 打开仓库 `Settings`
2. 进入 `Pages`
3. `Source` 选择 `Deploy from a branch`
4. `Branch` 选择 `main`
5. `Folder` 选择 `/ root`
6. 保存后等待 GitHub Pages 构建完成
